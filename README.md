我的AutoHotKey库，为了简化操作想了一些脚本。

## 切换虚拟桌面
[点击看源代码](./src/switch-desktop.ahk)

由于雷蛇鼠标驱动没有找到录制 Win快捷键 的，所以写了AHK去操作ctrl + win + 左右去切换虚拟桌面，方便工作桌面的分类和整理

操作指南：
通过`鼠标的侧键`操作左右切换。
